Asteroids_Vala
==============

Simple game using OpenGL

What´s this ?
-------------

An test of the darkcore lib . Darkcore uses opengl to display sprite.

The move the 'ship' with arrows ,  fire with space and 'w'.

If you hit all rock´s , the game ends.

Game has 4 level .
